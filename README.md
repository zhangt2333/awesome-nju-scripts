# Awesome-NJU-Scripts

Some scripts about the life at NJU. 与 NJU 日常相关的一些脚本，如自动健康填报、选课助手等，欢迎研究、使用、贡献 :smile:

# Latest

## 自动健康填报 `actions-NjuHealthReport`
* 自动进行每日健康填报
* 不需要自己购买服务器，也不需要自己配置服务器，真的 Serverless !!
